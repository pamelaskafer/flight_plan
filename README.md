## Script teórico para calcular um plano de vôo aerofotogramétrico

### Como rodar
```bash
python main.py 
```

### Variáveis
scale (cm)  
cam_focus_distance (mm)  
photo_size (cm)  
area (ha)  
plane_speed (km/h)  

### Saídas
1) Plane height (m)  
2) Airbase (m)  
3) Band distance (m e cm)  
4) Useful area (ha)  
5) Total of photos  
6) Total band  
7) Total of photos by band  
8) Photo interval (m/s)
